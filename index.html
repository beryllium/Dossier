Dossier

If your mid-size company is hitting the hurdle of remote workers with dozens of dynamic IP addresses requiring whitelisting to a multitude of EC2 Security Groups, you'll want to pay attention to the development of Dossier.

The plan is to start out small, using python, to create a command-line tool that can catalogue and modify security groups across multiple AWS regions and accounts. Eventually, the hope is to integrate this tool into workflows so that companies can automate the whitelisting (and subsequent pruning/ageing/removal) of dynamic IP addresses.
